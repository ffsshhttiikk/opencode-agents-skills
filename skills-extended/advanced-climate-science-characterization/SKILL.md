# Advanced Climate Science Characterization Skill

## Overview
This skill enables characterization in the domain of climate-science (earth-sciences). It represents research-level-level expertise and is designed for production use in research, industry, and educational contexts.

## Description
Use this skill when you need to perform characterization operations related to climate-science. This includes tasks such as:
- predict events
- interpret data
- analyze samples

The skill leverages field instruments and follows best practices established in the earth-sciences community.

## Trigger Conditions
This skill should be activated when:
1. The user explicitly requests characterization in the context of climate-science
2. The task requires research-level-level understanding of earth-sciences principles
3. The output needs to be hazard assessments
4. The work involves climate-science methodologies or techniques
